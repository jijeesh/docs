OpenStack

As of v0.5.0, RancherOS releases include an Openstack image that can be found on our releases page. The image format is QCOW3 that is backward compatible with QCOW2.

When launching an instance using the image, you must enable Advanced Options -> Configuration Drive and in order to use a cloud-config file.
